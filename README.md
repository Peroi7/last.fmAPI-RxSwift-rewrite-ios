# last.fmAPI-rewrite-ios-rx-ios 15

Search your favorite artists, albums, top tracks, photos, infos from last.fm API with Moya network layer including peristance on disk. 

*Required pod install: 
pod 'PureLayout'
pod 'Moya', '~> 15.0'
pod 'RxSwift', '6.5.0'
pod 'RxCocoa', '6.5.0'
pod 'RxCocoa', '6.5.0'
pod 'ProgressHUD'
pod 'SDWebImage'
